## What We Do

Bttr is a digital product agency that partners with organizations to deliver enterprise-grade platforms designed for scale, security, and regulatory compliance.

### Our Services

- **Design & Strategy** - Research-driven design that translates complexity into clarity
- **Engineering** - Full-stack systems built for performance, reliability, and longevity
- **Security & Compliance** - Regulatory-ready systems for healthcare, finance, and enterprise
- **Lifecycle Ownership** - Long-term system stewardship beyond launch

### Industries We Serve

- Aerospace & Defense
- Healthcare
- Financial Services
- Enterprise Technology

## Tech Stack

### Frontend
- Next.js / React / TypeScript
- Tailwind CSS
- Framer Motion

### Backend
- Python / FastAPI
- PostgreSQL / Redis
- Dramatiq (background jobs)

## Development

```bash
# Frontend
cd clients
pnpm install
pnpm dev

# Backend
cd server
uv sync
uv run task api
```
